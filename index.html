Cambrian Charter

A marine ROV project

Description

Designing and building spherical autonomous marine rovers for data collection, focussing on:

  • Climate data
    • Temperature
    • Salinity
    • Ocean currents
    • Pressure
  • Geological data
    • Topology
    • Bathymetry
    • Geomagnetic field
    • Local magnetic field interaction
  • Sonic data
    • Ultrasonic object detection
    • Ambient hydrophonic acoustics analysis
